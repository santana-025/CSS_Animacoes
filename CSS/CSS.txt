		 CSS

Tem 3 Forma de ser introduzido na página :

CSS inline = Diretamente na Tag (Tem Prioridade Nivel 1)
	<h1 style="collor: Red" > Trabalhando com o Css </h1>

CSS Interno = Antes do Fechamento do Heead (Só Serve na mesma Pág) (Tem Prioridade Nivel 2)
	<style>
    	H1 {collor : "Red";} (sempre usar o ponto e virgula)
	</style>

CSS Externo = Cria uma pasta com um arquivo css e chama ele pra
o codigo. Coloca antes do fechamento do head. (serve para todas as proximas pag) (Tem Prioridade Nivel 3)
	CSS / Estilo.css
Chamando o css = <link rel="stylesheet" href = "CSS/Estilo.css"

Pode se criar varios arquivos de css, por isso é de forma de cascata!
o mais usados é o css externo
